Hemuppgift 1

jag bad mina klasskamrater att surfa in på ett gammalt Mvc projekt som jag hade igång på min Webserver. och ta fem minuter på sig för att testa sidan Recepttips.se och sen svara på följande frågor:

Mina frågor

    Karin Jonsson svarade följande på frågan

  1. Hur upplever du recepttips.se som första gångs användare?

    2. Bra första intryck på förstasidan (men bildeffekterna var irriterande). Det kändes som en sida man skulle vilja använda, men som att den inte var färdig. Det var lite oklart hur man skulle göra och var man skulle klicka.

    Karin Jonsson svarade följande på frågan

  2. Vad kan vi förbättra på recepttips.se så att du får en bättre upplevelse?

    3. Ha texten framme hela tiden på de två bilderna på startsidan där man väljer mellan mat och dryck. Jag missade helt att man kunde välja det och såg det inte förrän tredje gången jag kom till startsidan. Design på sök ingrediens-sidan. Skulle vilja se sökrutan först, inte gömd längst ner. Dessutom större typsnitt och sökruta i ingredienser. Kanske en liten text som förklarar hur man söker. Jag tryckte först på sök-knappen i rutan "dina ingredienser", eftersom det var den som stack ut, men det gick inte så bra eftersom jag inte hade valt några ingredienser. Svårt att läsa texten på själva receptens sidor när texten ligger direkt på en bild. Kanske ändra något där?

    Karin Jonsson svarade följande på frågan

  3. Vad var dom tre bästa respektive mindre bra upplevelser från ditt besök på recepttips.se?

    4. Bäst

    1. Trevlig förstasida
    2. Användbar funktion att kunna söka på ingrediens
    3. Bra att kunna välja mellan fritextsökning och att klicka på ingredienserna

    Sämst

    1. Oklarhet om hur man skulle göra
    2. Sidan kände inte igen ingrediensen om man inte började med stor bokstav
    3. Inga kontaktuppgifter på kontaktsidan, bara ett formulär och en karta, känns oseriöst.

    Min Sammanställning

    Eftersom det bara var en i klassen som tog sig tid att svara på mina frågor så utgår jag ifrån det. Mina frågor var bra eftersom dom gav mig feedback på saker som kan vara bra att tänka på i framtiden och saker som jag själv inte har Reflekterat över. Sen att testpersonen i fråga är en utbildad webbutvecklare känner jag bara var positivt då denna personen lite mer än andra vet vad man skall leta efter och mer där till. Utifrån svaren jag fick kan jag dra slutsatsen att sidan inte är färdig för kund och det är inget jag inte kände till då detta var ett skolprojekt vi har lagt på is tillsvidare så svaren jag fick gick i linje med vad jag själv tyckte och tänkte. Fast några nya insikter så som att effekterna på första sidan var irriterande för användaren det var nytt. Något jag som utvecklare tyckte var coolt tyckte användaren var irriterande så där lärde jag mig något. Sen att textstorlek och lite andra saker var i fel format och på fel plats var också nyttig information. Något som jag funderar på att ändra nu efter denna kurs.
